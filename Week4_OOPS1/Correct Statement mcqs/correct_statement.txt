Problem statement
Figure out the correct statement

class Car{
    public : 
    int price;
};

int main() {
    Car c; 
    //Figure out the correct statement to set the price of the car object referred to by c to “10000”
}



Options: Pick one correct answer from below
• c->price = 10000
• Car.price = 10000
• c.price = 10000;

answer:-
c.price = 10000;
